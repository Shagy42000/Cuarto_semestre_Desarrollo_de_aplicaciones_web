# Galería Interactiva

## Descripción
Trabajo práctico de la Semana 5 sobre manipulación del DOM y manejo de eventos con JavaScript.

## Funcionalidades
1. **Agregar imágenes**: Mediante URLs o rutas locales
2. **Seleccionar imágenes**: Haz clic para seleccionar (solo una a la vez)
3. **Eliminar imágenes**: Elimina la imagen seleccionada
4. **Teclado**: Usa SUPR para eliminar la imagen seleccionada

## Instrucciones de uso
1. Ingresa una URL de imagen en el campo de texto
2. Haz clic en "Agregar Imagen" o presiona Enter
3. Haz clic en cualquier imagen para seleccionarla
4. Usa el botón "Eliminar" o la tecla SUPR para borrar la imagen seleccionada
5. Usa "Limpiar Galería" para borrar todas las imágenes

## Estructura de archivos
- index.html: Estructura principal
- styles.css: Estilos de la galería
- script.js: Lógica JavaScript
- assets/img/: Carpeta para imágenes locales (opcional)

## Tecnologías usadas
- HTML5
- CSS3 (Grid, animaciones)
- JavaScript (manipulación del DOM, eventos)

# Proyecto Semana 8: Bootstrap y JavaScript

Este proyecto es una implementación de una página web responsiva utilizando Bootstrap 5 y JavaScript para interactividad.

## Características principales

1. **Encabezado con Navbar**: Menú de navegación responsivo con 4 enlaces.
2. **Carrusel de imágenes**: Implementado con Bootstrap Carousel.
3. **Tabla de productos**: Tabla estilizada con datos ficticios de productos.
4. **Formulario de contacto**: Con validación en tiempo real y contador de caracteres.
5. **Botón de alerta**: Muestra alertas personalizadas con JavaScript.
6. **Pie de página**: Con información de contacto y enlaces a redes sociales.

## Tecnologías utilizadas

- HTML5
- CSS3 (con estilos personalizados)
- JavaScript (ES6)
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)

## Funcionalidades JavaScript

1. **Validación de formulario**:
   - Campos obligatorios: nombre, email y mensaje.
   - Validación en tiempo real.
   - Mensajes de error específicos.
   - Contador de caracteres para el mensaje.

2. **Interactividad**:
   - Botón de alerta personalizada.
   - Smooth scroll para enlaces internos.
   - Resaltado de sección activa en navbar.

3. **Responsividad**:
   - Diseño adaptativo para móviles, tablets y escritorio.
   - Componentes Bootstrap responsivos.
